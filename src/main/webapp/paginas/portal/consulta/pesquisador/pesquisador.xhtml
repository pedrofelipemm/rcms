2011 - 2012
Elaboração de Material Didático Interdisciplinar de Apoio ao Processo de Desenvolvimento de Software nas Etapas de Modelagem e Implementação

Descrição: A presente proposta tem como objetivo a elaboração de material didático interdisciplinar que atenda às etapas de modelagem e implementação de sistemas de software. Pretende-se elaborar material didático que envolva a integração de disciplinas de programação, desenvolvimento de Software e Banco de Dados ministradas nos módulos do curso Sistemas de Informação. Essas disciplinas nortearão o desenvolvimento de um material didático que apoiará os professores e alunos ao longo do curso.
Situação: Em andamento; Natureza: Pesquisa.
Alunos envolvidos: Graduação: (4) / Mestrado acadêmico: (2) / Doutorado: (1) .

2009 - 2012
Software Reuse

Descrição: Motivados pela importância da Tecnologia da Informação, visando contribuir para o avanço do conhecimento e da tecnologia no desenvolvimento de aplicações na WEB 2.0, em Engenharia de Software, e com foco nas idéias sobre a Reutilização de Software, este projeto visa pesquisar e explorar conceitos, técnicas, métodos e ferramentas para tornar o Reuso de Software uma prática intra e inter comunidades. Com foco na reutilização, investiga-se uma Estratégia para o Desenvolvimento de Software, que visa o compartilhamento de recursos e a cooperação na web, apoiando e melhorando a Internet como plataforma de trabalho. O principal objetivo deste projeto é elaborar uma Estratégia para o Desenvolvimento de Software, orientada para Reuso, combinando idéias, disciplinas, tecnologias, e ferramentas, conforme o paradigma emergente da WEB2.0. Outros objetivos são: Definir um conjunto de recursos computacionais, incluindo ontologias, serviços, componentes, padrões, frameworks, sistemas multi-agentes, e ferramentas, para apoiar a Estratégia Orientada para o Reuso; Desenvolver uma Linha de Produtos de Software em domínios de aplicações na WEB 2.0. Essas linha de produtos servirá como estudo de caso para propor, testar, e refinar a Estratégia de Reuso proposta; Elaborar documentações e tutoriais que instruem sobre o PDS orientado para o Reuso; e Elaborar publicações que divulgam os resultados alcançados. A Estratégia servirá para orientar o desenvolvimento orientado para o reuso de software de aplicações na web. O apoio computacional produzido com o projeto pode ser utilizado para facilitar a prática da estratégia nas empresas. Por fim, a documentação do projeto, tutoriais e publicações servirão para divulgar e estimular o uso da estratégia pelas diferentes comunidades que desenvolvem software..
Situação: Em andamento; Natureza: Pesquisa.
Alunos envolvidos: Doutorado: (2) .
